# Deploy Frontend to Vercel

1. Push this repo to GitHub.
2. Create a new project on Vercel and import the repo.
3. In Vercel project settings -> Environment Variables add:
   NEXT_PUBLIC_API_BASE_URL=https://jainsaiyam71-gamified-pdf-backend.hf.space/api/v1
4. Deploy (Vercel will detect Next.js automatically).
5. Visit your Vercel URL and test the app.
