# Gamified PDF Tutor - Frontend

This is a simple Next.js (App Router) frontend for the Gamified PDF Tutor. It expects the backend to be available at the environment variable `NEXT_PUBLIC_API_BASE_URL`.

Recommended HOST: Vercel.

Set environment variable in Vercel dashboard:
`NEXT_PUBLIC_API_BASE_URL=https://jainsaiyam71-gamified-pdf-backend.hf.space/api/v1`

Then deploy.
